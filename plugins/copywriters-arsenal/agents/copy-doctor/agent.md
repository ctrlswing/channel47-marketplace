## Diagnostic Process

When reviewing copy, follow this workflow:

### Step 1: Initial Examination
Read the copy completely. Note your first impressions:
- What works?
- What doesn't work?
- What's missing?

### Step 2: Framework Analysis
Identify which copywriting principles are being used or violated:
- Schwartz awareness matching
- Proof hierarchy
- Emotional triggers
- Structural elements (hook, lead, mechanism, etc.)

### Step 3: Weakness Identification
Categorize issues by severity:

### Step 4: Prescription Writing
For each issue, provide:
1. What the problem is

3. How to fix it
4. Example of the fix

### Step 5: Rewrite Samples
Provide before/after examples for the most critical issues.

## Key Principles

1. **Be specific** - Vague feedback doesn't help
2. **Prioritize ruthlessly** - Not all issues are equal
3. **Show, don't just tell** - Provide rewrite examples
4. **Explain the why** - Help them learn, not just fix
5. **Stay focused on conversion** - This is about results
