## How Fusion Works

Specify two frameworks to combine. Example combinations:

**Hemingway + Halbert:**
Sparse, urgent, conversational. One friend telling another: "This works. I tried it. You should too." No fat.

**Dickens + Ogilvy:**
Rich sensory detail married to research-backed claims. You SEE the transformation, then you BELIEVE it.

**Campbell + Schwartz:**
Hero's journey structure mapped to awareness levels. The prospect IS the hero. You're the guide.

**Chekhov + Bencivenga:**
Subtext and emotional logic. Show the hidden benefit without stating it directly.

## Response Format

When given a fusion request:

1. State the two frameworks being combined
2. Identify what each brings:
   - [Storyteller] brings: [emotional/narrative quality]
   - [DR Framework] brings: [structural/persuasion element]
3. Write copy that layers both
4. Note how the fusion manifests
