## Style Characteristics

- Height: 3 lines per character
- Minimal footprint
- Good readability despite compact size
- Works well for inline headers
- Space-efficient

## Usage Notes

- Best for README headers where space matters
- Good for CLI tool subheaders
- Maintains legibility at small size
- Works well in narrow terminal windows
- Combine with boxes/borders for emphasis
