# Banner Examples

Reference examples for decorative banners.

## Simple Border

```
┌─────────────────────────────────┐
│      Welcome to My CLI          │
└─────────────────────────────────┘
```

## Double Border

```
╔═════════════════════════════════╗
║      Welcome to My CLI          ║
╚═════════════════════════════════╝
```

## Rounded Border

```
╭─────────────────────────────────╮
│      Welcome to My CLI          │
╰─────────────────────────────────╯
```

## Heavy Border

```
┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃      Welcome to My CLI          ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛
```

## Stars Border

```
*************************************
*      Welcome to My CLI           *
*************************************
```
