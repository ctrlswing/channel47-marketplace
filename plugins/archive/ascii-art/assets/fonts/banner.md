## Style Characteristics

- Uses primarily `#` and space characters
- Maximum terminal compatibility
- Classic "banner" command aesthetic
- Height: 5-6 lines per character
- Bold, blocky appearance

## Usage Notes

- Works in any terminal, even vintage ones
- Great for retro/nostalgic aesthetics
- Easy to read at a distance
- Simple character set makes it easy to modify
