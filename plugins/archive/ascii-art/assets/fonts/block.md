## Block Characters Reference

| Character | Description |
|-----------|-------------|
| █ | Full block |
| ▀ | Upper half block |
| ▄ | Lower half block |
| ▌ | Left half block |
| ▐ | Right half block |
| ░ | Light shade |
| ▒ | Medium shade |
| ▓ | Dark shade |
| ╔╗╚╝ | Double line corners |
| ═║ | Double lines |

## Usage Notes

- Height: 6 lines per character
- Best for: Splash screens, headers, logos
- High visual impact
- May not display correctly in all terminals
- Requires Unicode support
