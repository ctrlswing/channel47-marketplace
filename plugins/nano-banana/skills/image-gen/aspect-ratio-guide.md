# Aspect Ratio Guide for Nano Banana Pro

## Quick Reference

| Ratio | Pixels | Best For |
|-------|--------|----------|
| 1:1 | 1024x1024 | Instagram, profile pics, thumbnails |
| 16:9 | 1920x1080 | YouTube, presentations, desktop |
| 9:16 | 1080x1920 | Stories, Reels, TikTok, mobile |
| 21:9 | 2520x1080 | Cinematic, ultrawide displays |
| 4:3 | 1365x1024 | Classic photos, documents |
| 3:4 | 1024x1365 | Portraits, posters |
| 2:1 | 2048x1024 | Panoramas, website headers |

## Use Case Recommendations

### Professional/Business
- **Presentations**: 16:9
- **Business Cards**: 3.5:2 (custom)
- **Brochures**: 4:3 or 3:4
- **Website Headers**: 2:1 or 3:1

### Creative/Art
- **Portfolio Pieces**: Match original intent
- **Prints**: Standard photo ratios (4:3, 3:2)
- **Posters**: 2:3 or 3:4
- **Album Covers**: 1:1

### E-commerce
- **Product Photos**: 1:1 (universal)
- **Hero Banners**: 16:9 or 2:1
- **Category Images**: 4:3

### Marketing
- **Ads (Display)**: 1:1, 16:9, 9:16
- **Email Headers**: 2:1
- **Blog Posts**: 16:9

## Composition Tips by Ratio

### Square (1:1)
- Center the subject for impact
- Use symmetry effectively
- Good for single subjects
- Works well with patterns

### Landscape (16:9)
- Use rule of thirds horizontally
- Great for scenes with depth
- Lead the eye across the frame
- Ideal for environments

### Portrait (9:16)
- Vertical leading lines
- Stack elements top to bottom
- Full-length subjects
- Text overlay friendly

### Cinematic (21:9)
- Dramatic horizontal scenes
- Panoramic landscapes
- Epic scale compositions
- Film-like atmosphere

### Classic (4:3)
- Traditional photography feel
- Balanced compositions
- Documentary style
- Versatile for many subjects

## Technical Notes

### Resolution by Model

**Flash Model (up to 1024px):**
- 1:1 → 1024x1024
- 16:9 → 1024x576
- 9:16 → 576x1024

**Pro Model (up to 4K):**
- 1:1 → 2048x2048
- 16:9 → 3840x2160
- 9:16 → 2160x3840

### Upscaling Considerations
- Generate at highest available resolution
- Use external upscalers for larger sizes
- Pro model produces better upscaling base
