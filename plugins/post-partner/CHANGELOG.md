# Changelog

All notable changes to Post Partner will be documented in this file.

## [1.0.0] - 2024-12-28

### Added
- Initial release
- `/post` skill for guided post brief creation
- Adaptive conversation based on input type (dense, sparse, draft)
- Support for Claude Code logs, voice memo transcripts, drafts, and rough ideas
- Take question framed by conversational temperature (claim, observation, question, provocation)
- Brief artifact with structured components (insight, audience, take, entry point)
- Graceful exit for "no post here" scenarios
- Multiple thread detection with notes for unpursued angles
- Conversation guide for consistent tone (peer, not coach)
- Brief template for artifact structure
