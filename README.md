# Channel 47 Marketplace

Battle-tested Claude Code plugins from daily workflows.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add ctrlswing/channel47-marketplace
```

## Available Plugins

- [Creative Writing](https://channel47.dev/plugins/creative-writing) - Writing assistant for editing and improving prose
- [Google Ads](https://channel47.dev/plugins/google-ads) - Query Google Ads data using GAQL with OAuth authentication
- [Prompt Enhancer](https://channel47.dev/plugins/prompt-enhancer) - Optimize prompts using Anthropic's 2025 best practices

## Philosophy

- **Publisher, not aggregator**: Only tools built and maintained by Channel 47
- **Quality over quantity**: Every plugin is battle-tested in daily use
- **Open source first**: Building for long-term impact and community
